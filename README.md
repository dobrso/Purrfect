# Purrfect - Сервис для заботливых хозяев

## Обзор продукта
Purrfect - хаб для владельцев домашних животных, где можно заказать вещи для питомца, записаться к ветеринару, писать или читать полезные статьи про животных,
прогнозировать будущие траты при помощи ML-модели и так далее.

## Функциональные возможности
* Возможность прочтения или написания статей
* Возможность заказать вещи для питомца (своего рода маркетплейс)
* Возможность записаться к ветеринару
* Возможность получить онлайн-консультацию по поводу питомца
* Прогнозирование будущих трат при помощи ML-модели
* Список будет пополняться по мере появления новых возможностей

## План работ
* Спринт 1. Разработка ядра (фундамента) сайта
* Спринт 2. Интеграция ML-модели
* Спринт 3. Доработка сайта

## Технологический Стек
* Язык программирования: Python 3.12+
* Backend (Api): Django Rest Framework, Django Cors Headers
* Frontend: HTML3, CSS5, JavaScript, React
* Тестирование: Pytest
